# FlightInformationAPI

## Running the API
1. Open the solution in Visual Studio 2022.
2. Set the `FlightInformationAPI` project as the startup project.
3. Press `F5` to run the application.
4. Access the Swagger UI in your browser. It will open automatically, and you can access through navigating to https://localhost:<port>/swagger/index.html (replace <port> with your applicationâ€™s port number). For example, it might be https://localhost:7244/.

## Running Tests

1. Open the `FlightInformationAPI.Tests` project in Visual Studio 2022.
2. Right-click on the test project and select "Run Tests" to execute the tests.
